Git相关

Posted on 2022-12-08 Edited on 2022-12-09

就让这个随着自己使用git每次用到的指令慢慢更新吧！

git init：初始化一个git仓库（实际为创建一个名为.git的目录）

git clone [url] directoryname：克隆一个现有的git仓库，并自定义名称

git status：查看当前文件状态

git add filename或git add *：把文件提交到暂存区

git commit -m "提交信息"：把暂存区的文件提交到本地仓库，记得之前看一下git add了没有！！！

git push：把本地仓库提交到远程仓库
